# makebuzz

Make some buzz by generating some clickbaity headlines, BuzzFeed-style. This application utilizes BuzzFeed's RSS feeds to get seed
headlines and build Markov chains to generate new sentences.

This application powers [@BuzzFeed_IsHip](https://twitter.com/BuzzFeed_IsHip)


### Notes

The `headlines.json` file has some Spanish/Portuguese/French headlines in it since I didn't filter those out originally. Those are sometimes interesting,
but if you don't want to see those then I suggest you remove them.
